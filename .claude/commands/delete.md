포스트를 찾아서 삭제한다.

## 실행

1. 검색어로 포스트 찾기 (제목 또는 파일명)
2. 여러 개 매칭되면 목록 보여주고 선택 요청
3. 삭제 전 포스트 제목과 요약 표시
4. 사용자 확인 후 삭제 실행
5. 관련 이미지 파일이 있으면 함께 삭제 여부 확인

## 인자

- $1: 검색어 (제목 일부 또는 파일명)

검색어가 없으면 최근 포스트 목록을 보여주고 선택하게 한다.

## 주의사항

- 삭제 전 반드시 확인 받기
- git에서 완전히 제거하려면 커밋 필요
