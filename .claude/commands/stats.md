블로그의 전체 통계를 보여준다.

## 수집 항목

1. **포스트 현황**
   - 전체 포스트 수
   - 발행된 포스트 수
   - 초안(draft) 포스트 수

2. **태그 통계**
   - 사용된 태그 목록
   - 태그별 포스트 수 (상위 10개)

3. **카테고리 통계**
   - 카테고리별 포스트 수

4. **최근 활동**
   - 최근 작성/수정된 포스트 5개
   - 첫 포스트 날짜
   - 마지막 포스트 날짜

## 출력 형식

```
## 블로그 통계

### 포스트
- 전체: X개
- 발행: X개
- 초안: X개

### 태그 (상위 10개)
1. 태그명 (X개)
2. ...

### 카테고리
- 카테고리명: X개

### 활동
- 첫 포스트: YYYY-MM-DD
- 최근 포스트: YYYY-MM-DD
```
