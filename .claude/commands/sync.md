# 포스트 동기화 확인

현재 작업 내용과 관련된 포스트를 확인하고 업데이트 또는 새 포스팅을 제안한다.

## 수행 작업

1. **최근 변경사항 분석**
   - `git diff` 또는 `git log`로 최근 작업 내용 파악
   - 변경된 파일, 추가된 기능, 수정된 설정 등 확인

2. **관련 포스트 탐색**
   - `content/posts/` 폴더의 모든 포스트 스캔
   - 작업 내용과 관련된 키워드, 태그, 주제 매칭
   - 시리즈로 연결된 포스트 확인

3. **분석 결과 제시**

   다음 형식으로 결과를 보여준다:

   ### 관련 포스트
   - 포스트 제목과 파일 경로
   - 관련성 설명

   ### 권장 행동
   - **업데이트 필요**: 어떤 내용을 추가/수정할지 구체적으로 제안
   - **새 포스트 제안**: 주제, 예상 내용, 관련 시리즈 여부
   - **포스팅 불필요**: 이유 설명

4. **사용자 확인 후 실행**
   - 업데이트 또는 새 포스트 작성 여부를 사용자에게 확인
   - 승인 시 해당 작업 수행

## 사용 예시

```
/sync
```

최근 작업을 기반으로 분석한다.

```
/sync Hugo 설정 변경
```

특정 작업 내용을 명시하면 해당 내용을 기준으로 분석한다.
