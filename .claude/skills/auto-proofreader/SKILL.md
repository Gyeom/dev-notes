# Auto Proofreader Skill

포스트 작성 완료 후 자동으로 기본 문체 검사를 수행한다.

## 검사 항목

### 필수 수정
1. `:` 로 끝나는 문장 → 마침표로 수정
2. `~입니다`, `~합니다` → `~이다`, `~한다`로 수정

### 경고
1. "다음과 같다", "아래와 같이" 표현
2. "~할 수 있다", "~하면 된다" 남발
3. 과도한 강조 (정말, 매우, 굉장히)
4. 언어 명시 없는 코드 블록

## 동작

1. 작성된 포스트 내용 검사
2. 필수 수정 항목 발견 시 자동 수정
3. 경고 항목은 사용자에게 알림
4. 심각한 문제 없으면 조용히 통과
