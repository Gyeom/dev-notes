# 이력서 동기화 Skill

"이력서", "resume", "about" 관련 요청 시 자동으로 실행된다.

## 원본 파일

- **이력서 원본**: `content/about.md`
- **PDF**: `resume.pdf` (about.md에서 수동 export)

## 동기화 규칙

### 포스트 → 이력서 연동

about.md에서 블로그 링크가 있는 경우:
1. 포스트 제목이 변경되면 about.md도 업데이트
2. 포스트가 삭제되면 about.md에서 링크 제거 또는 수정

### 확인 항목

| 항목 | 위치 |
|------|------|
| 블로그 링크 | `<a href="/dev-notes/posts/...">` |
| 성과 설명 | `.achievement-detail` |
| 핵심 역량 | Core Competencies 테이블 |

## 사용 예시

```
사용자: 이력서 업데이트해줘
→ about.md 확인 후 변경된 포스트와 동기화

사용자: resume 수정해야 할 것 같은데
→ about.md가 원본임을 안내하고 수정 진행
```

## 체크리스트

- [ ] 블로그 링크가 유효한지 확인
- [ ] 포스트 제목과 이력서 설명이 일치하는지 확인
- [ ] 날짜가 맞는지 확인 (포스트 파일명의 날짜)
