## Dev Notes에 오신 것을 환영합니다

이 블로그는 개발하면서 배운 것들을 기록하고 공유하기 위해 만들어졌습니다.

### 주요 주제

- 개발 TIL (Today I Learned)
- 프로젝트 정리
- 기술 문서
- 트러블슈팅 기록

### 자동 포스팅

이 블로그는 Claude와 GitHub Actions를 활용하여 자동으로 문서를 생성하고 배포합니다.

```bash
# 새 포스트 생성 예시
./scripts/new-post.sh "포스트 제목" "태그1,태그2"
```

앞으로 유용한 개발 노트들을 많이 공유하겠습니다!
